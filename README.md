# Ensamblador y Procesador RISC-V

Este proyecto incluye la implementación de un **ensamblador** y un **procesador monociclo** basado en el conjunto de instrucciones **RISC-V 32I**.

## Descripción
- **Ensamblador RISC-V**: Herramienta para convertir código assembly de RISC-V a código máquina.
- **Procesador Monociclo RISC-V32I**: Implementación hardware del conjunto básico de instrucciones RISC-V (RV32I) en un diseño monociclo.
